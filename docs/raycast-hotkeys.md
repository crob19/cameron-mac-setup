# Raycast Hotkeys

Reference for manually re-configuring Raycast on a new Mac.
Raycast has no hotkey import/export API, and its config lives in an encrypted
local SQLite DB tied to Keychain — so these have to be set by hand in
Raycast Settings → Extensions (per-command hotkey column).

| Hotkey    | Action                                   |
